API Endpoint

Python 
# pip install azure-ai-vision-imageanalysis
from azure.ai.vision.imageanalysis import ImageAnalysisClient
from azure.ai.vision.imageanalysis.models import VisualFeatures
from azure.core.credentials import AzureKeyCredential

client = ImageAnalysisClient(
    endpoint=endpoint,
    credential=AzureKeyCredential(key))

result = client.analyze_from_url(
    image_url=image_url,
    visual_features=[VisualFeatures.CAPTION, VisualFeatures.TAGS, VisualFeatures.READ])

print(f"Caption: {result.caption.text}")
for tag in result.tags.values:
    print(f"Tag: {tag.name} ({tag.confidence:.0%})")
C# 
// dotnet add package Azure.AI.Vision.ImageAnalysis
using Azure;
using Azure.AI.Vision.ImageAnalysis;

var client = new ImageAnalysisClient(
    new Uri(endpoint),
    new AzureKeyCredential(key));

ImageAnalysisResult result = client.Analyze(
    new Uri(imageUrl),
    VisualFeatures.Caption | VisualFeatures.Tags | VisualFeatures.Read);

Console.WriteLine($"Caption: {result.Caption.Text}");
foreach (var tag in result.Tags.Values)
    Console.WriteLine($"Tag: {tag.Name} ({tag.Confidence:P})");
cURL 
curl -X POST "{endpoint}/computervision/imageanalysis:analyze?api-version=2024-02-01&features=tags,caption,read" \
  -H "Content-Type: application/json" \
  -H "Ocp-Apim-Subscription-Key: {key}" \
  -d '{"url": "https://example.com/image.jpg"}'
Exam Tip: The features query parameter accepts a comma-separated list. You can request multiple features in a single call.

SDK Usage

Python 
# pip install azure-ai-formrecognizer
from azure.ai.formrecognizer import DocumentAnalysisClient
from azure.core.credentials import AzureKeyCredential

client = DocumentAnalysisClient(
    endpoint=endpoint,
    credential=AzureKeyCredential(key))

poller = client.begin_analyze_document_from_url(
    "prebuilt-invoice", invoice_url)
result = poller.result()

for doc in result.documents:
    total = doc.fields.get("InvoiceTotal")
    if total:
        print(f"Invoice Total: {total.content}")
C# 
// dotnet add package Azure.AI.FormRecognizer
using Azure;
using Azure.AI.FormRecognizer.DocumentAnalysis;

var client = new DocumentAnalysisClient(
    new Uri(endpoint),
    new AzureKeyCredential(key));

AnalyzeDocumentOperation operation = await client
    .AnalyzeDocumentFromUriAsync(
        WaitUntil.Completed,
        "prebuilt-invoice",
        new Uri(invoiceUrl));

AnalyzeResult result = operation.Value;
foreach (var document in result.Documents)
{
    if (document.Fields.TryGetValue("InvoiceTotal", out var totalField))
        Console.WriteLine($"Invoice Total: {totalField.Content}");
}
cURL 
curl -X POST "{endpoint}/formrecognizer/documentModels/prebuilt-invoice:analyze?api-version=2024-11-30" \
  -H "Content-Type: application/json" \
  -H "Ocp-Apim-Subscription-Key: {key}" \
  -d '{"urlSource": "https://example.com/invoice.pdf"}'
Exam Tip: prebuilt-read = OCR only. prebuilt-layout = OCR + tables/structure. prebuilt-document = OCR + key-value pairs. Know the difference.

Choose Decision Tree — Which Service to Use?

ScenarioService
Read text from imagesAI Vision OCR (Read)
Read text from documents / formsDocument Intelligence
Classify images into categoriesCustom Vision (Classification)
Find objects with bounding boxes in imagesCustom Vision (Detection) or AI Vision (objects)
Analyze video content (faces, topics, keywords)Video Indexer
Count people in live video feedSpatial Analysis
AI Vision OCR vs Document Intelligence: Use AI Vision Read for simple text-in-image extraction. Use Document Intelligence when you need structured data (tables, key-value pairs, specific document types like invoices).
Custom Vision vs AI Vision Objects: Use AI Vision objects for general object detection (no training). Use Custom Vision when you need to detect domain-specific objects with your own training data.
Video Indexer vs Spatial Analysis: Video Indexer analyzes recorded/uploaded video for insights. Spatial Analysis processes live camera feeds in real time on edge devices.
