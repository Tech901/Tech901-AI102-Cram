Traps Exam Traps & Distinctions

temperature vs top_p: Change ONE, not both
frequency_penalty vs presence_penalty: frequency scales by repetition count; presence is a flat penalty if token appeared at all
Content filter responses: Prompt blocked → 400, Completion blocked → 200 with content_filter
Severity levels: 0, 2, 4, 6 (even numbers only, NOT 0-3 or 1-4)
fieldMappings vs outputFieldMappings: fieldMappings = BEFORE enrichment, outputFieldMappings = AFTER enrichment
API key headers: api-key for OpenAI, Ocp-Apim-Subscription-Key for all others
Contributor vs User: Contributor = manage resources (NOT call APIs), User = call APIs (NOT manage)
S3 HD vs L-series: S3 HD = multi-tenancy (many indexes), L-series = storage-optimized (few indexes, huge data)
Semantic search: Re-ranks top 50 BM25 results only
Custom Vision export: Only compact domains can be exported (TensorFlow, CoreML, ONNX, Docker)
Query type default: Simple query is default, not full Lucene
searchMode default: "any" (OR), not "all" (AND)
DALL-E 3: n=1 only, URL expires in 60 minutes
Custom Translator: Needs 10,000 parallel sentences minimum
CLU tiers: Standard = English-only, Advanced = multilingual (neural)
QnA chit-chat: 5 personalities — Professional, Friendly, Witty, Caring, Enthusiastic
SSML break durations: "strong" = 1000ms, "x-strong" = 1250ms
Speaker Verification vs Identification: Verification = 1:1, Identification = 1:N (up to 50 candidates)
