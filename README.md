## Recommended Study Plan

1. Read the **Exam Overview** to understand domain weights and the service selection decision tree
2. Study one domain sheet per session — don't cram everything at once
3. Take the **domain test** after each study session to check retention
4. Use the **SDK Reference** and **Portal Workflows** sheets for hands-on review
5. When you score **70%+ on each domain test**, take the **Full Practice Exam**
6. Review the **Glossary** for any terms you're unsure about

## About the Exam

- **Exam:** AI-102 — Microsoft Azure AI Engineer Associate
- **Passing score:** 700/1000 (approximately 70%)
- **Format:** ~50 questions, multiple question types, ~100 minutes
- **Exam retires:** June 30, 2026

## Tips

- Each practice test question includes an explanation and a link back to the relevant cram sheet
- Tests can be shuffled and retaken — questions are randomized each attempt
- The full exam randomly samples 50 questions from the 138-question pool, so every attempt is different
- All content works offline — save the page or clone this repo for local access
- Print-friendly: use your browser's print function to create PDF study sheets
