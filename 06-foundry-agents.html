Three Runtime Components

ConversationDurable session, stores items (messages, files)

Execution Modes

Streaming: stream=True — real-time token delivery.
Background: background=True — async processing, poll for results.

Patterns Multi-Agent Orchestration

PatternDescription
SequentialAgents in order, output passes forward to next agent
ConcurrentParallel execution, results aggregated
MagenticManager coordinates specialists dynamically
Handoff / SwarmHand off based on expertise matching
Human-in-the-loopPause execution for human input / approval
Exam Tip: Sequential is simplest; Magentic allows the manager agent to dynamically decide which specialist to invoke. Handoff passes full conversation context to the next agent.

Deploy Deployment Options

Deployment TypeRegionBillingKey Feature
Global StandardAny regionPay-per-tokenHighest available quota
StandardSingle regionPay-per-tokenData stays in region
Data Zone StandardWithin US or EUPay-per-tokenData residency compliance
Provisioned typesSingle regionReserved PTUPredictable latency
Batch typesVaries~50% discountAsync, 24hr target SLA
PTU (Provisioned Throughput Units): Reserved capacity purchased in advance. Best for steady, high-volume workloads needing guaranteed latency.
Global Standard: Routes requests to any available region automatically. Highest quota but data may cross regions.
Data Zone Standard: Keeps data within US or EU boundary — use when data residency is required but exact region is flexible.
Exam Alert: Know which deployment type to choose for data sovereignty (Data Zone Standard), cost optimization (Batch), and latency-sensitive apps (Provisioned).

Code SDK Quick Reference

Create & Run an Agent

Python 
from azure.ai.projects import AIProjectClient
from azure.identity import DefaultAzureCredential

client = AIProjectClient(
    credential=DefaultAzureCredential(),
    project="your-project-connection-string"
)

# Create agent
agent = client.agents.create_agent(
    model="gpt-4o",
    name="memphis-311-agent",
    instructions="You are a Memphis 311 assistant.",
    tools=[{"type": "code_interpreter"}]
)

# Create conversation and add message
conversation = client.agents.create_conversation()
client.agents.create_message(
    conversation_id=conversation.id,
    role="user",
    content="What potholes were reported today?"
)

# Get response
response = client.agents.create_response(
    conversation_id=conversation.id,
    agent_id=agent.id
)
C# 
using Azure.AI.Projects;
using Azure.Identity;

var client = new AIProjectClient(
    new Uri("your-project-connection-string"),
    new DefaultAzureCredential()
);

// Create agent
var agent = await client.Agents.CreateAgentAsync(
    model: "gpt-4o",
    name: "memphis-311-agent",
    instructions: "You are a Memphis 311 assistant.",
    tools: new[] { new CodeInterpreterToolDefinition() }
);

// Create conversation and add message
var conversation = await client.Agents
    .CreateConversationAsync();
await client.Agents.CreateMessageAsync(
    conversationId: conversation.Value.Id,
    role: "user",
    content: "What potholes were reported today?"
);

// Get response
var response = await client.Agents
    .CreateResponseAsync(
        conversationId: conversation.Value.Id,
        agentId: agent.Value.Id
    );

Streaming Response

Python 
# Stream tokens as they arrive
with client.agents.create_response(
    conversation_id=conversation.id,
    agent_id=agent.id,
    stream=True
) as stream:
    for event in stream:
        if event.type == "content.delta":
            print(event.delta, end="")
C# 
// Stream tokens as they arrive
await foreach (var update in
    client.Agents.CreateResponseStreamingAsync(
        conversationId: conversation.Value.Id,
        agentId: agent.Value.Id))
{
    if (update is ContentDeltaUpdate delta)
    {
        Console.Write(delta.Text);
    }
}

tools = [
